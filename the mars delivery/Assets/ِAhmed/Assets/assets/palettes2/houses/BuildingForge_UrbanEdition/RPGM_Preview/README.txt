This example map uses default RPG Maker MZ tiles (specifically Outside_A2,Outside_A4 and Outside_A5 tilesets) for demonstration purposes.

These default tiles are NOT included in the Full Version asset pack.

To view or use the example map properly, you must own a licensed copy of RPG Maker MZ. 
Users without a valid license are not permitted to use these RTP assets in any form.

Files included on Full Version :
  • Map001.json          – example city block using BuildingForge Urban tiles
  • MapInfos.json        – map registration data
  • Tilesets.json        – tileset slot definition
  • System.json          – project configuration
  • Custom PNGs in img/  - (A3,B,C,D)
